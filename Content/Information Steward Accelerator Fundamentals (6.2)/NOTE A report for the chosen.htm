NOTE:A report for the chosen ISA Rule Binding ID must have been run in ISA before the staging table can be built. dspCompose creates a staging table in the Staging Data Source database. The column names in the table match the column names listed on the Report Data Viewer page for the associated ISA report. In some cases, the staging table may need to be rebuilt, for example, if bindings on the report in IS are changed. Clear the staging table by clicking Clear Staging Area, then click Build Staging Area.
