NOTE:Any field in the table that does not allow NULLS, such as a Primary Key, automatically is a Hard Required field.

Development properties such as Column Properties can be added to control data entry for an Excel File. The Column Properties added to an Excel File are available for all Requests based on the initial Template. For example, a list box is added to an Excel file for a Template. All Excel Files for any Requests created using the Template will contain the list box.

The Validate button automatically displays on the Request page if a List Box, Check Box or a Required Column are added to the Data Entry page. This is configurable on the Vertical View of the Template on the Advanced tab. The defaults these settings is on the Parameters under Configuration.
