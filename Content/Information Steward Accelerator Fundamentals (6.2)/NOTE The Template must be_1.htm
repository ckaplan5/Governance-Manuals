NOTE:The Template must be generated and not active to add an ISA External Request Scenario.

To register the rule:

  1. Click the Browser Back button twice to return to the Templates page.
  2. Click ISA External Request Scenarios .
  3. Enter Customer Informal Word in Rule ID field.
  4. Enter KNA1_TRNXX - NAME1 in Rule Binding ID field.
  5. Enter Update Customer Informal Word TrainingXX - #Data# in Request Description Format field.
  6. Select SAP from Target ERP System list box.
  7. Click the Save button, the Vertical View displays.
  8. Select ISA Training from the ISA Project ID list box.
  9. Select Customer Name has Informal Word for TrainingXX/KNA1_TRNXX – NAME1 from ISA Rule Binding ID list box.
  10. Click the Save button.
