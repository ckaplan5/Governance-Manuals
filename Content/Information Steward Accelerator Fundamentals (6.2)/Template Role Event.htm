Template Role Event

  1. Self-Directed – Generate the Request and Review

To review the configurations:

  • Activate the Template
  • Generate the request from ISA.
  • Review the Data Entry page for the column properties.
  • Edit NAME1 for one of the records.
  • On the Vertical View of the Data Role in the Request, click the Download Request Data to review the column properties on the Excel Request.
  • Verify the Review Role’s Approval page properly displays the Comparison Approval.
