NOTE:If rules were just added to IS, the IS repository in Collect must be refresh before the rules can be added to a Project Distribution in ISA.

There are two ways to add rules:

  • Add rules individually from a drop down list.
  • Based on Quality Dimensions as they display in IS.

To add rules to a Distribution from a list box:

  1. Locate Customer Master – Training XX in the Dependent pane.
  1. Click Rules in the Dependent pane.
  2. Click the Add button.
  3. Select Customer NAME2 Missing for TrainingXX from the Rule ID list box.
  4. Click the Save button.
