5.22 Initiate the Request and Verify Data Entry Configurations

Once all the Template configurations have been done they need to be tested and verified they are correct. Verify the Data and Manufacturing configurations are correct by creating an Excel Initiated Request.

To generate the Excel Initiated Request:

  1. The Email To in the email must be external contact created under configuration in dspCompose.
  2. The Subject of the email must exactly match the Subject Alias configure for the External Request Scenario.
  3. An Excel spreadsheet with three rows based on the staging file configured on the External Request Scenario must be attached.
  4. The names of the worksheets within the spreadsheet must match the staging sheet configured on the External Request Scenario.

Once the properly created email is sent a series of validations occur:

  1. The email poller processes based on the External Data Email address configured in dspCompose.
  1. dspCompose locates the sender email based on the correct email address for the individual External Request Scenario.
  2. It checks the email for the alias in the subject line that corresponds to a subject alias configured on an external request scenario that creates Excel-initiate requests.
  1. It checks the Worksheet name to an external request scenario. If there is a match between the worksheet name in the Excel file and the name entered in Sheet Name, dspCompose™ copies the request data to a staging table for the external request scenario’s Template.
  1. dspCompose™ then maps the data in the columns in the worksheet to the template columns according to mappings created on the Template (External Request Scenario Columns) page.

If any of the validations fail, dspCompose sends an email back to the sender with a description of the failure.

Once all the validations pass, the Request is created and the user assigned to the first role in the workflow receives an email notification indicating a request is ready to be processed.

To test the configurations:

  1. Open Mozilla Thunderbird on the desktop.
  2. Click Write button to create a new email.
  3. Enter admin@Training.net in the To field.
