NOTE:The Template must be generated in order to add column properties.

There are two methods to create column properties:

  • Generate a Request and then add column properties on the data entry page in dspCompose.
  • Add the column properties in the DSP System Administration. Navigate to WebApps > cMass_Data > Pages search for the Template pages.

Add column properties to disable and make required KUNNR and D0110, NAME1 needs to be enabled and required.
