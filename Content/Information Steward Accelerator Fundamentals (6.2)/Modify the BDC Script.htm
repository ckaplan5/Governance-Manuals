5.5 Modify the BDC Script

BDC screen fields are individual fields that capture data during the BDC script recording. If redundant steps were created during a recording or if a step was left out, the screens can be manually added, deleted or edited without having to re-record the script, regardless of the recording mechanism used.

The Screens from the recording display with a fields icon at the top of the page. The number of screen fields display on the Fields icon. The fields for the selected screen display in the Dependent pane. If Is Field is checked, the record represents a field in SAP and can be mapped to data in a process. The Template must be inactive to edit the BDC screens.

Modify the BDC Script to add the Old Material Number field that was missed during the recording.

To add a field:

  1. Click Vertical View for the Template.
  1. Click the Configuration tab.
  1. Click BDC Screen to view the recording.
  1. Highlight the Screen to view the fields for the recording.
