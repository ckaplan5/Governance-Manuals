GUI Scripts

  • Should be the last resort when creating a script.
  • Launches the SAP GUI and records each action the user takes. Recording must be stopped.
  • Before recording a GUI script, the SAP Logon Pad must be opened.
  • While recording multiple transaction codes in one Template is supported with GUI scripting it is not recommended. The linking of Templates together is achieved via a Process
  • GUI scripts can be debugged using a debugging script.
