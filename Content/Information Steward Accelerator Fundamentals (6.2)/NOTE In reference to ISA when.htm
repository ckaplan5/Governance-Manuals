NOTE:In reference to ISA, when refreshed, it extracts the IS metadata. If any changes are made in IS during development, the ISA repository in Collect needs to be refreshed.

  • dspCompose™ - A workflow-enabled component of the ISA to govern the remediation, review and approval of failed data from an ISA report. Users are assigned to roles, which are associated with Templates. A Template aligns with a single BDC or GUI script, an Integrate Template, or a custom Template, and can be reused for multiple requests. A request drives the workflow process to fix failed data and post to SAP. Requests are automatically generated from a report in ISA.
  • Integrate – Is one method dspCompose uses to post data to SAP. Integrate uses standard SAP posting functionality as well as custom processing to post data to SAP. Integrate loads data to SAP using:
  • Batch Data Communication (BDC) processing
  • Graphical User Interface (GUI) scripting
  • A Remote Function Call (RFC) or a BAPI
  • Generated Flat files

NOTE:Data Services is the other method to load data into SAP.
