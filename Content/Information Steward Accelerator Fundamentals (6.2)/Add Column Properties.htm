5.17 Add Column Properties

Column properties ensure that specific values are selected through a list box or combo box when manually entering data or specifying required fields. If the data for the request is imported from a view and the value of the field does not meet the criteria of the column property, the record displays as invalid.

Add column properties disable and make MATNR required. Add a combo box to the MATKL and a list box to EXTWG columns.

List Box or a Combo Box are Control Types within Column Properties that display a field as a list with options from a selected Table or View. Column Properties are specified to control the behavior of the Control Types, what is displayed and how values are stored in a table.

When creating List or Combo Box Views:

  • Use the naming convention webXXXList where XXX describes what is being listed, e.g., webProductList.
  • Include only the columns that apply to the List Value, List Display, List Select or List WHERE clause registrations.
  • Include a unique set of data.
  • Do not sort in the view. Sorting is done in the DSP using the List Order By Column Property.

Combo boxes are recommended when there is a larger set of data. The use can start typing the value needed and the list will filter based on the value entered.

Column Properties can be added in System Admin as demonstrated in Exercise 4 or they can be added directly on the Data Entry page. The benefit of adding them on the Data Entry page is that the changes can be tested immediately to ensure there are no errors.

To add Column Properties:

  1. Locate MaterialMasterUpdateXX page.
  2. Click Column Properties .
  3. Click the Add button.
  4. Select RMMG1-MATNR from the Column list box.
  5. Select Text Box from the Control list box.
  6. Select Disabled from the Control Status list box.
  7. Click the Save button, the Vertical View displays.
  8. Select Yes (hard) from the Required list box.
  9. Click the Save button.
  10. Select MARA-MATKL from the Column list box.
  11. Select Combo Box from the Control list box.
  12. Click the Save button, the Vertical View displays.
  13. Select webMATKLList from the List Source list box.
