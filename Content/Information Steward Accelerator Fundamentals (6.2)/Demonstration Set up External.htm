5.7 Demonstration – Set up External Email Account

NOTE:The following was designed for reference only. There are no steps to perform.

An external request scenario creates an Excel-initiated request, an external contact sends an email with an Excel file (with one or multiple worksheets) attached to an external data email account configured for dspCompose™.

A dedicated email account is added on the External Data Email Accounts page. Once the scenario has been configured, a valid user will send an email to this address to add a request. dspCompose™ will poll that account for emails from users who are assigned the External Data role.

To set up an External Data Email (Demonstration):

  1. Click Configuration on the Navigation pane to expand.
  2. Click External Data Email Accounts on the Navigation pane.
  1. Click the Add button if records exist on the page.
  2. Enter email server address in Host field.
