Project Distribution

3.4 Add User to Project Distribution

Any users registered in ISA can be added to a Distribution, however users are not active by default.

Users who belong to a Distribution can receive workflow emails about the rules associated with the selected Project Distribution and can generate a request once it is configured on the Template.

For the purpose of training the user has already been added to the Project Distribution.

To add user to a distribution:

  1. Click Users for Customer Master – TrainingXX Distribution.
  2. Enter TrainingXX in the filter field.
  3. Click the Enter key.
  1. Verify Active check box to add user.
