NOTE:The Template Name must be unique. The Template Name can contain A-Z, 0-9, and underscore. No special characters are allowed in template names and Integrate will replace them with underscores when the Template is saved.

  1. Enter Update Material Master in the Description field.
  2. Select SAP from the SAP Connection ID list box.
  3. Select BDC Script from Type list box.
  4. Click the Save button, the Vertical View displays.
  5. Enter MM02 in Transaction Code list box, a message displays.
