NOTE:The Template must be active to import into dspCompose.

  1. Create a Template in dspCompose

Create a new Template in dspCompose and import the Integrate Template.

To create a Template:

  1. Click the BackOffice Site link.
  2. Click dspCompose on the Navigation pane.
  3. Click Team on the Navigation pane.
  4. Locate Material Master Team.
  5. Click Create Template .
  6. Click Create from Integrate Template.
  7. Enter MaterialMasterUpdateXX in Template Name field.
  8. Select SAP from the Connection list box.
  9. Select MaterialMasterXX from Integrate Template ID list box.
  10. Click the Save button.
  11. Click Create Template , a message displays.
  12. Click the OK button.
