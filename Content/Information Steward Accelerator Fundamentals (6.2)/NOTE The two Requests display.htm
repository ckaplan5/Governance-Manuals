Templates are created based on an ISA Report to generate a Request to fix failed data and post to SAP. dspCompose is connected to a DSP component called Integrate. Integrate contains the loading mechanisms to push data into SAP.

Integrate loads data to SAP using:

  • Batch Data Communication (BDC) processing
  • Graphical User Interface (GUI) Scripting
  • A Remote Function Call (RFC) or BAPI

Integrate can also create and transfer user defined text files formatted as Delimited, Fixed Width, or XML to the SAP server.

Integrate uses standard SAP posting functionality as well as BackOffice delivered custom processing and can be used by any platform component (including the framework) to post data to the target system.
