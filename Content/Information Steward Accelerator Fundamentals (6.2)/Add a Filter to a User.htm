5.0 Add a Filter to a User

Filters are customized at the project distribution level to limit data contained in rules. Project Distribution filters apply to all rules that include the specified field(s).

For example, a distribution level filter could be configured to send a sales report to all plants at a summary level, while another filter could be configured to send a report with data for a specific plant.

User filters can be combined with project distribution filters. When multiple user or project distribution filters are applied across different columns, these filters are combined using the AND operator. If multiple user or project distribution filters are applied to the same column, these filters are combined using the OR operator. If a user filter and a distribution filter are applied to the same column, the two filters are combined using the AND operator, not the OR operator. A validation warning will trigger in the corresponding filter page if this scenario would occur.
