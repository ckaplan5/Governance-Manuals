5.23 Self-Directed – Verify the Template Configurations

Verify the configurations made to the template are correct.

  1. For the Data Role:
  • The Data Role is the only Role enabled when the Request is created.
  • A workflow email generated
  • Verify the Manufacturer Part Number and Manufacture Number fields are disabled on the Data Entry page for that Role only.
  • Click Edit on the Data Entry page to verify the column properties are configured correctly.
  • Download the Request Data to verify the column properties and column controls copied to the Excel spreadsheet.
  • Validate and Finish the Data Role.
  1. For the Manufacturing Role
  • Becomes enabled once the Data Role is finished.
  • A workflow email generated
  • Verify the Manufacturer Part Number and Manufacture Number fields are the only enabled fields on the Data Entry page.
  • Using the Mass Change feature, enter a value in the Manufacturer Part Number for all records.
  • Download the Request Data to verify the column properties and column controls copied to the Excel spreadsheet.
  • Click the Validate button for the Manufacturing Role to verify validations fail for the missing Manufacturer Part Numbers, an error message displays with 9 errors.
  • Click the Failures button to view the Validation Comments.
  • Click the Finish button for the Manufacturing Role, a message displays, click OK.
