NOTE:For BDC recording, SAP opens in learning Mode using the defined transaction code, which utilizes the same recording mechanism as the SHDB transaction code in SAP to track the modified fields touched by a user during the recording. Enter credentials if prompted.

  1. Enter trainingXX in Back End Password field.
  1. Enter trainingXX in Confirm Back End Password field.
  1. Click Authenticate button; SAP opens and logs into the MM02 transaction code.
  1. Enter LNDXX01 in Material field.
