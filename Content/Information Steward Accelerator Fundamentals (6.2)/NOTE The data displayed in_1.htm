  1. Create a Template

Once the rule is imported into ISA. A Template needs to be created in dspCompose™ based on the ISA report so the data can be remediated by the business user. Templates are based on a recording in SAP.

dspCompose supports multiple SAP connections that are configured in Integrate. dspCompose uses Integrate™ to post to SAP.

The Template containing the SAP recording can be reused for multiple requests and be configured for more complicated changes by importing it from Integrate into dspCompose and associating it with a new Template.

By default, only the user creating the Template has access to the Template. Other users can be added by the creator of the Template. Once created, a Template is like a mini WebApp. Each Template has its own users and can be customized by adding data entry controls and validations to ensure the data is business ready prior to posting to SAP. Customizations drive the way in which the end user interacts with a data entry page.

When creating Templates, it is important to look at all the different rules to see if one Template recording can be used for multiple rules. This will help minimize the number of recordings needed. As demonstrated in Exercise 3, multiple rules can be added to one Template. If data controls are unique to a Rule, a new dspCompose Template can be created based on the same Integrate Template.

Create a new Template in dspCompose based on the Missing_Cust_Address_FieldsXX Template which is a copy of the Template used in the first and third exercises in this manual. The Template will be imported from Integrate. The new Template will be configured specifically for the Informal Word Rule by defining data controls and validations to ensure the data is business ready prior to posting to SAP.

To create a Template:

  1. Click the BackOffice Site Link.
  1. Click dspCompose in the Navigation pane.
  1. Click Team in the Navigation pane.
  1. Click Create Template for the Customer Master Team.
  1. Click Create from Integrate Template.
  1. Enter Customer Informal Word XX in Template Name field.
  1. Select SAP from Connection list box.
