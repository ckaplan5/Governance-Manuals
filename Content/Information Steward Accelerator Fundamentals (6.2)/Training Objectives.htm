Training Objectives

Upon completion of this course, users will be able to:

  • View Reports based rules from Information Steward
  • Understand Requests are based on a reusable Template
  • Generate Requests based on the Report
  • Understand data controls are defined to manage data entry to ensure records are business ready
  • Understand Templates are fully customizable resulting in the requests are unique based on the Template
  • Understand requests are workflow enabled
  • Update data using Mass Change Feature
  • Download data to Excel for updating
  • Upload updated data from Excel
  • Review the updated data
  • Compare the updated data with the current data in SAP
  • Approve and reject data
  • Post the data
  • Understand the Project import process using Collect
  • Understand the creation of Distributions
  • Understand the process and methodology of adding users and rules to a Distribution.
  • Create user filters and determine how the filters affect user data.
  • Add additional Roles to workflow
  • Configure ISA External Scenario
  • Configure External Request Scenarios
  • Configure Excel Level Column Controls
  • Configure column controls per role
  • Add Column Properties to control data entry
  • Add Page level Validation Rules to control data entry
  • Configure Template and Template Role Rules
  • Configure Request Role Rules
  • Configure Finish Table Refresh
  • Configure Comparison Approval
  • Create Template and Role Validations
  • Understand Basic Integrate™ navigation
  • Understand how to view and remediate failed data using ISA and dspCompose
